This program loads, edits, and saves a playlist using a singly linked list.
Below is a simple explanation of the main functions:
1. remove_newline_if_exists
Removes the ending newline character from a line of text if it exists.
2. load_file
Opens the playlist input file.
Reads each line as a song title.
Creates a new list item for each song.
Adds these items to the playlist.
3. save_file
Opens the output file.
Writes each song from the playlist into the file.
4. print_tracks
Shows all songs in the playlist on the screen, each with a track number.
5. main
Loads the playlist from the file.
Displays the loaded tracks.
Removes the 4th song.
Inserts a new song at position 3.
Saves the updated playlist to the output file.
